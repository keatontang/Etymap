## Getting started

Install Livereload [extension](http://livereload.com/extensions/) and extract `data.7z`.

```shell
pip3 install -r requirements.txt
yarn
yarn start dev
# or
yarn start prod
```

The development flavour is available on [localhost:5000](http://localhost:5000).
